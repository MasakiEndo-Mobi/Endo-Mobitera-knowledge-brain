# 80_sandbox — 個人 raw 層

未整理の着想・思考メモ・外部リサーチ要約・議論ログを置く場所。

## 構造

| フォルダ | 用途 | thinking_status |
|---|---|---|
| `inbox/` | 未分類の着想（1 アイデア = 1 ファイル） | `spark` |
| `ideas/` | 着想・戦略メモ・提案 | `developing` |
| `research/` | 外部記事の要約・評価レポート | `developing` |
| `discussions/` | 議論ログ・対話記録 | `developing` |
| `drafts/` | canonical 昇格候補（L0/L1/L2 構造を持つ） | `ready_to_propose` |
| `assets/` | 派生物（HTML / 画像 / SVG 等） | — |

## 昇格フロー

```
inbox/ → ideas/ or research/ or discussions/ → drafts/ → canonical (10_projects/* etc.)
```

詳細は [[AGENTS]] §2 を参照。
