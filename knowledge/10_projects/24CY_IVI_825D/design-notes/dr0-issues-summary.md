IVI 825DのDRBFM書類の修正点を表形式でまとめます。

| シート名/項目            | 修正内容                        | 備考                                                |
| ------------------ | --------------------------- | ------------------------------------------------- |
| **全体**             | 部品名を統一                      | ESC/フロントESC、COVER GLASS/ガラス、TFT BRACKET/TFTブラケット等 |
| **②変更点・変化点一覧**     | 対象外としているが対象になっている項目を修正      | -                                                 |
| ↑                  | BT ANTに図を添付                 | ④心配点シートにある図を添付                                    |
| ↑                  | BONDING ASSYの「━」を修正         | 「対象外DRBFM対象、除外理由」を記載                              |
| ↑                  | BTアンテナ(構造/組立/形状/ケーブル長)の欄を修正 | 「ランク」「DRBFM No.」を「━」から変更(対象外ではない)                 |
| **③機能の着眼点**        | 左上の表の品名とトヨタ品番を修正            | おかしい内容を訂正                                         |
| ↑                  | 仕向を修正                       | 「北米とカナダ」から正しい内容に変更                                |
| **④心配点抽出**         | 耐衝撃性の表現を修正                  | 「疲労破壊」→「破壊」                                       |
| ↑                  | ESC No.4の記載を追加              | DRBFM対象だが記載が無い                                    |
| ↑                  | リアカバーNo.7-9の記載を追加           | DRBFM対象だが記載が無い                                    |
| ↑                  | リアカバーの番号ズレを修正               | 両面テープがNo.7まである                                    |
| ↑                  | TFT BKTに400Dの厚みを追加          | 220Dと825Dの厚みが入っている                                |
| **⑤DRBFM(共通)**     | 発生度、影響度、検出度、危険度を追記          | 自動算出、他機種参考                                        |
| ↑                  | 特殊特性を追記                     | -                                                 |
| ↑                  | 作成/調査/承認を入力                 | -                                                 |
| ↑                  | 担当期限欄の空欄を記入                 | 散見される空欄を埋める                                       |
| ↑                  | 「仮金型試作品」を「PP0-1試作品」に変更      | 全ての部品で記法変更                                        |
| **825D MM 14inch** | ガラス割れ(シャープエッジ)を故障モードに追記     | 耐衝撃を含んでいるため                                       |
| ↑                  | 補強板金厚みを追記                   | -                                                 |
| ↑                  | インパネ衝撃を評価項目に追記              | 設計にて対策済み                                          |
| ↑                  | セット仕様書の50N有無を記載             | 正式版発行後に記載                                         |
| ↑                  | {#検証}の内容を明確に記載              | 何を意味するか明示                                         |
| ↑                  | 沿面距離の情報を追記                  | 参考機種400Dより長いことを記載                                 |
| ↑                  | 剛性確保の解析結果を補足                | 209Mpaは母体220Dと同等構造で問題なし                           |
| **ESC**            | 担当期限のズレを修正                  | 一番上の行など                                           |
| ↑                  | 耐衝撃性の故障モードを修正               | 「疲労破壊」→「破壊」                                       |
| ↑                  | 爪嵌合削除の理由を明記                 | 仮固定用途のため削除、ビス締結に置き換え                              |
| ↑                  | キシミ音防止の確認                   | 樹脂/樹脂以外でも摺動確認が必要                                  |
| **両面テープ**          | 900Bの面積を確認して修正              | 確認数値の根拠を明確に                                       |
| ↑                  | 面積相違の説明を追記                  | 解析で最適化を図っている旨を記載                                  |
| ↑                  | 耐衝撃性の計算結果を記載                | 解析未実施のため衝撃計算を追記                                   |
| ↑                  | 熱衝撃の確認方法を修正                 | 1SではなくPP0-2で確認                                    |
| ↑                  | 全体の貼り付け面積の妥当性を追記            | 過去機種比較で問題ない旨を記載                                   |
| **リアカバー**          | 耐衝撃の故障モードを修正                | 「疲労破壊」→「破壊」                                       |
| ↑                  | 歪みに対する強度の記載を追加              | ESC部分のクリップ締結についても追記                               |
| **前板**             | BT ANT固定の記載を修正              | 「参考機種○○と同等」に変更                                    |
| **BTアンテナ**         | 仮固定の記載を修正                   | 825Dでは全てフックでクランプ                                  |
| ↑                  | クランプ配置の記載を簡潔化               | 「適切に配置」に変更                                        |
| **FFC/FPC**        | 全長の記載を確認                    | FFCとFPCで違うため統一                                    |
| **設計基準チェックリスト**    | DR1欄の○を空欄に戻す                | G-Wing2、BOM反映前のため                                 |
| ↑                  | M-90の遵守確認資料を修正              | G-Wing2登録対象外のため誤記修正                               |
| ↑                  | M-95に15W以下である旨を記載           | 禁止内容の判別を明確化                                       |
| ↑                  | M-105を修正                    | 「◎」→「○/当該機種で確認」(プレス部品新設のため)                       |
