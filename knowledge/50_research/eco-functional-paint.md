## 概要

| 項目   | 内容                                   |
| ---- | ------------------------------------ |
| 技術名称 | 環境対応型塗料・機能性塗料                        |
| 対象分野 | 車載機器、加飾部品、工業製品全般                     |
| 技術目的 | CO₂削減、SDGs対応、機能性付加（透過性、リサイクル適合、耐傷性等） |
| 主な方式 | 植物由来塗料・UV硬化型塗料の採用                    |
| 対応課題 | 石油由来原料依存、塗装工程の省エネルギー、リサイクル困難         |

## 詳細

- **環境対応型塗料（バイオペイント）**
  - 原料の約25%を植物由来に置換
  - 既存の塗装設備（スピンドル/ロボット）で使用可能
  - 石油系塗料と同等の高外観品質を維持

- **機能性塗料（透過系）**
  - ブラックアウトなどの要求に対応する、透明性と隠蔽性を両立したナノオーダーの顔料設計
  - メタリックや黒色でも透過可能な特殊塗料

- **機能性塗料（リサイクル系）**
  - 熱可塑性樹脂の混入によるリサイクル樹脂の脆弱化を防止
  - UV硬化型により、塗膜形成時の熱反応を不要化

- **UV硬化型塗料**
  - 「耐傷」「耐指紋」性能を持ち、秒単位で硬化可能
  - 省スペース・省エネルギー化とCO₂排出削減を両立

## 想定Q＆A

**Q1. UV硬化型塗料は耐候性に課題はないのですか？**  
A1. 改良されたUV硬化型塗料は、耐候性試験においても熱硬化型と同等以上の性能を示しています。

**Q2. なぜ植物由来塗料がSDGsに貢献するのですか？**  
A2. 石油資源の使用を減らすと同時に、再生可能なバイオマスを利用することでカーボンニュートラルに近づくためです。

**Q3. リサイクル樹脂が脆くならない理由は？**  
A3. 熱可塑性樹脂を使用せず、UV硬化型で常温固化させるため、リサイクル工程での干渉が抑制されます。

**Q4. 既存ラインでの置換は本当に簡単ですか？**  
A4. はい、既存のスピンドルやロボット塗装機でも使用可能な粘性・乾燥性に調整されています。

## Before/After

| 状態 | Before（従来技術） | After（当該技術） |
|------|------------------|------------------|
| 環境負荷 | 石油系塗料100%使用、CO₂多排出 | 植物由来25%、CO₂削減、省エネ塗装 |
| 塗装装置 | 高温加熱装置が必要 | UV照射装置で瞬時硬化、省スペース化 |
| リサイクル性 | 塗膜の熱可塑性が再生材料を脆化 | UV硬化型によりリサイクル適合性向上 |
| 外観品質 | 加飾の自由度は高いが一部制限 | 高透明性と隠蔽性を両立しデザイン自由度拡張 |
