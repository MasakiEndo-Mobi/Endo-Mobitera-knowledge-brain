# Onyx FDM 部品の表面平滑化レポート

## ① 要点箇条書き

- **目的**: Markforged **Onyx** (カーボン短繊維入りナイロン) 製 FDM 部品を塗装下地に適した表面粗さ **Ra ≤ 10 µm** に仕上げ、かつ寸法公差 **±20 µm** を維持する。
- **基準工法１**: パテ塗布→室温硬化→手研磨 (タクト *45 min*, コスト *≈1 300 円/個*, Ra≈0.8–1 µm※)。
- **基準工法２**: プライマー＋サーフェス塗布（２～３コート）（タクト 75～105min, コスト *≈1 500 円/個*, Ra≈0.2–0.3 µm※）
- **既存技術を組み合わせた 6 工法 (A–F)** を比較し、タクト・コストともに基準工法より有利な案を探索。
- 追加要件として「新規性」「コスト性」「品質」を★5 で評価し、誰でも理解できるように専門用語へ脚注を付与。

※ コストは、部品表面積 0.5 m²として試算。

> **★評価基準**  - 新規性: 業界での採用例の少なさと技術革新度。コスト性: 直接コストが基準工法よりどれだけ低いか。品質: 達成できる表面粗さの優秀さ。

### 工法の詳細

- **A IR+フォーム酸**: SLS ナイロン向け実績が豊富で、ひと工程で Ra を大幅に改善。薬液管理が容易。
- **B クライオタムリング**: 低温脆化させて突出部のみ除去。液体窒素コストがランニング要因。
- **C ウェットブラスト+CO₂**: 微細アルミナで湿式研磨し、CO₂ 雪で脱脂と静電気除去を同時に行う。
- **D 超音波+UV 樹脂**: 超音波キャビテーションで低粘度樹脂を浸透させ、UV で瞬時硬化し孔埋め。
- **E レーザーリメルト**: ファイバレーザーを高速走査して表面を再溶融し自己平滑化。
- **F MR ポリッシング**: 磁場で硬化した磁性流体にナノ砥粒を懸濁し、極低荷重で均一に削られ、Ra 0.05 µm の鏡面が得られます。磁場を弱めれば流体はすぐ低粘度に戻り、循環して再利用できます。

| **IR 予熱**<br>(旧KB埋め込み: Pasted image 20250724095034.png\|200)                | 赤外線ヒーターは光の一種で物体表面に直接エネルギーを届けます。電熱炉のように空気全体を温めるのではなく、樹脂が光を吸収して即座に発熱するので立ち上がりが早く、省エネです。Onyx 部品を 60 ℃程度に均一予熱すれば溶剤蒸気が馴染みやすくなり、短時間で表面がなめらかになります。赤外線は非接触なので部品を傷つけず、形状が複雑でもムラなく加熱できるのが強みです。 |
| **フォーム酸ヴェイパースムージング**<br>(旧KB埋め込み: Pasted image 20250724095439.png\|200)<br> | フォーム酸は食酢にも含まれる弱酸で、ナイロンをほんのわずかに溶かす性質があります。密閉槽で加熱すると酸が蒸気になり、部品の表面を薄く溶かして「自己平滑化」を起こします。溶けた分子は表面張力でくぼみを埋め、数分でテカリのある仕上がりに。蒸気処理後は酸が気化してほぼ残らず、水洗いで無害化できるため環境負荷も比較的低い方法です。                   |
| **クライオタムリング**<br>(旧KB埋め込み: Pasted image 20250724095138.png\|200)            | タムリングは小さな部品を回転させながらメディアと擦り合わせるバレル研磨の一種です。これを液体窒素で -80 ℃以下に冷却して行うと、樹脂はガラスのように脆くなり、バリや積層の山だけが砕け落ちます。母材は硬い繊維で補強されているので主要寸法は保たれ、研磨熱も出ません。後処理の温度復帰も数分で済み、寸法変化が小さいのが利点です。                  |
| **薄膜パテ**<br>(旧KB埋め込み: Pasted image 20250724095158.png\|200)                 | パテは塗料より粘度が高い充填材です。微小な孔や段差に刷り込んで乾燥硬化させ、最後に軽く研磨すれば平坦面が得られます。本工程では UV 硬化型アクリルパテを用い、紫外線ランプで 30 秒以内に固めるためタクトを短縮。塗膜は 20 µm 程度と薄く、後の塗装膜厚と合算しても許容範囲に収まるよう配慮しています。                            |
| **ウェットマイクロブラスト**<br>(旧KB埋め込み: Pasted image 20250724095610.png\|200)         | 5 µm ほどの極小アルミナ粒子を水と一緒にノズルから噴射し、霧状に当てて削る方法です。水膜が熱と静電気を奪うので樹脂が溶けたり焦げたりしにくく、粉じんも飛散しません。微粒子が表面の凸部だけを柔らかく削り、Ra が大きく低減します。使用後のスラリーは沈降タンクで粒子を回収し再利用できるため、ランニングコストも抑えられます。                   |
| **CO₂ スノー洗浄**<br>(旧KB埋め込み: Pasted image 20250724095626.png\|200)            | 液体 CO₂ をノズルで急減圧するとドライアイスの雪 (スノー) になります。雪粒がぶつかると昇華して体積が 500 倍に膨張し、その風で異物を吹き飛ばすと同時に低温で静電気を除去。水分を全く残さず部品を速乾状態にできるため、直後の塗装でハジキや泡を防げます。洗浄後にゴミが残りにくく歩留まり向上に寄与します。                          |
| **超音波キャビテーション**<br>(旧KB埋め込み: Pasted image 20250724095640.png\|200)          | 液体中に 40 kHz 付近の超音波を入れると「キャビテーション」と呼ばれる微小な泡が高速で生まれ、潰れる衝撃波が表面の汚れや未融合孔を拡げます。同時に浸透性の高い低粘度樹脂を溶液に混ぜておくと、泡の空洞に樹脂が吸い込まれ、孔が埋まります。気泡を止めたあと UV ランプで一気に硬化させれば平滑かつ密実な皮膜が得られます。                    |
| **UV 樹脂浸透硬化**<br>(旧KB埋め込み: Pasted image 20250724095702.png\|200)            | 紫外線で瞬時に硬くなるアクリル樹脂を使用します。低粘度なので毛細管現象で微細孔に流れ込みやすく、樹脂同士の化学結合で密着力が高いのが特徴。硬化収縮が小さい配方を選べば寸法変化も抑えられます。従来の熱硬化パテと比べ 1/10 以下の時間で硬化し、ラインタクトの短縮に大きく寄与します。                                        |
| **ファイバレーザー表面リメルト**<br>(旧KB埋め込み: Pasted image 20250724100110.png\|200)       | シングルモードのファイバレーザー(波長 1 µm)をガルバノミラーで高速走査し、樹脂表面の 10 µm 程度だけを再溶融させます。溶けた層は表面張力で自己平坦化し、冷えて固まると積層痕が目立たなくなります。非接触・マスクレスなので複雑形状にも適用でき、照射パラメータを変えることで粗研磨と仕上げを 1 工程で切り替え可能です。                  |
| **N₂ パージ**<br>(旧KB埋め込み: Pasted image 20250724100127.png\|200)               | レーザー照射中に窒素ガスを吹き付け、酸化や黄変を防ぐ工程です。ナイロンは高温で酸素に触れると黄色くなるため、無酸素雰囲気にすることで色の変化や表面脆化を抑制します。同時にガス流でスモークや微粒子を排出し、レーザービームを安定させる効果もあります。                                                          |
| **磁性流体ポリッシング (MRF)**<br>(旧KB埋め込み: Pasted image 20250724100145.png\|200)     | 磁性流体は鉄系ナノ粒子を含むオイルで、磁場をかけると瞬時に粘度が上がります。流体内にナノサイズの砥粒(ダイヤなど)を混ぜ、磁石コア周辺に集めると「柔らかいバフ」状になります。部品がこの流体層に触れると低荷重で均一に削られ、Ra 0.05 µm の鏡面が得られます。磁場を弱めれば流体はすぐ低粘度に戻り、循環して再利用できます。                  |
| **スラリー循環システム**<br>(旧KB埋め込み: Pasted image 20250724100206.png\|200)           | 研磨中に砥粒や削り粉が混ざると効率が落ちるため、MRF ではフィルターと撹拌機を組み合わせた循環系を使います。フィルターでゴミを除去しつつ、撹拌で砥粒を均一に分散させることで毎回同じ研磨力を維持。流体と砥粒を使い捨てないのでコストと環境負荷を同時に下げられるのが強みです。                                             |

## ③ 今後のアクション

1. **見積もりシート**: 本表のタクト・コストを Excel に転記し、手研磨比の削減率を算出。
2. **実験計画 (DoE)**: A→C→B→D→E→F の順でラボ試験を行い、Ra・公差・塗装密着を測定。
3. **FMEA** 作成: 各工程で失敗モードと歩留まり≧95 % を確認。
4. **スケールアップ**: 300×300×200 mm 実サイズへの適用試験を実施し、量産可否を判断。
