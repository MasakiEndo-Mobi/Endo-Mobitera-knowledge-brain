# 3Dプリンターフィラメント吸水性

## 概要
| 材料                  | 吸水性の特徴                                                         |
| --------------------- | -------------------------------------------------------------------- |
| ナイロン11 (PA11)     | 少量吸水で環境安定性に優れる                 |
| ナイロン12 (PA12)     | PA6より吸水が少なく、湿潤環境向けに適合         |
| CF強化ナイロン         | 低湿気感度かつ高機械特性（PA612-CF、PAHT-CF） |
| CNF複合フィラメント   | 耐加水分解性・寸法安定性に優れる    |
| その他（PEEK, PP, PETGなど） | 高性能樹脂や非吸水性材料の採用増加                                   |

## 詳細
- **ナイロン11 (PA11)**  
  - SLS向けパウダーでは極めて少量の水分しか吸収せず、光・熱・化学薬品にも強い。  
  - 環境安定性が高く、精密部品や医療用プロトタイプにも利用。

- **ナイロン12 (PA12)**  
  - PA6と比べて吸水率が低く、湿潤環境下でも寸法変化が抑制される。  
  - 耐湿部品やスナップフィット製品での採用が拡大中。

- **CF（Carbon Fiber）強化ナイロン**  
  - ナイロンマトリックスに短繊維を添加し、寸法安定性と剛性を向上。  
  - PolymakerのPA612-CFはPA6/66より低湿度感度を実現。  
  - BambuLabのPAHT-CFはPA12+CFで低吸水かつ高耐熱を両立。

- **CNF（セルロースナノファイバー）複合フィラメント**  
  - CNF自体は親水性だが、SEBSとの複合により耐加水分解性と寸法安定性が向上。  
  - ノズル摩耗が少なく、環境負荷低減にも寄与する。

- **その他のフィラメント**  
  - **PEEK/PEKK**：吸水率＜0.1%で航空宇宙・医療用途に最適。  
  - **PP（ポリプロピレン）**：ほぼ非吸水で耐化学性に優れる。  
  - **PETG/PC**：中程度の吸水性だが、乾燥処理で品質安定可能。

## 今後の展望
- 表面コーティングや界面改質技術の導入により、吸水抑制型複合フィラメントの開発が加速。  
- PEEKやPEKK、グラフェン複合など超高温樹脂・ハイブリッド材料の普及で、湿潤環境対応部品の適用領域が拡大。  
- バイオ由来材料やCNF改質技術による、持続可能かつ低吸水性のバイオ複合フィラメント研究が活発化。  
- 産業用プリント機の品質安定化とコストダウン進展により、湿度管理を最小化した高精度造形ソリューションが実用化へ。
