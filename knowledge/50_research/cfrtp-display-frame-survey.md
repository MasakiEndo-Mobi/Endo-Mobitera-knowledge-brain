# CFRTP製 車載ディスプレイフレーム 技術調査レポート

## 1. 目的

本レポートは、炭素繊維強化熱可塑性プラスチック（CFRTP）を車載ディスプレイフレームに適用する際の技術的課題と可能性を調査し、今後の開発方針を明確にすることを目的とする。

## 2. CFRTPの概要

- **特徴:**
  - 軽量かつ高強度・高剛性
  - 高い設計自由度
  - リサイクル性
  - 優れた耐衝撃性
- **課題:**
  - コストが高い
  - 成形技術が複雑
  - 導電性を持つため、電気的絶縁が必要

## 3. 車載ディスプレイフレームへの適用

### 3.1. メリット

- **軽量化:** 車両の燃費向上に貢献
- **薄型化:** デザイン性の向上
- **高剛性:** 振動による表示のブレを抑制
- **安全性:** 衝突時のエネルギー吸収

### 3.2. 課題と対策

| 課題 | 対策 |
|:---|:---|
| **コスト** | - 量産効果によるコストダウン<br>-ニアネットシェイプ成形による後加工の削減 |
| **成形** | - ハイブリッド成形技術の導入<br>- シミュレーションによる成形条件の最適化 |
| **電気的絶縁** | - 絶縁コーティング<br>- 絶縁シートの挿入 |
| **電磁波シールド** | - 導電性フィラーの添加<br>- 導電性コーティング |

## 4. 競合技術との比較

| 材料 | メリット | デメリット |
|:---|:---|:---|
| **アルミニウム** | - 高い熱伝導性<br>- 優れた電磁波シールド性 | - 重い<br>- 加工コストが高い |
| **マグネシウム** | - 軽量<br>- 高い振動吸収性 | - 腐食しやすい<br>- 発火のリスク |
| **樹脂（ガラス繊維強化）** | - 安価<br>- 成形性が良い | - 強度・剛性が低い<br>- 熱に弱い |

## 5. 今後の展望

CFRTPは、コストや成形技術の課題を克服できれば、車載ディスプレイフレームの材料として非常に有望である。特に、電気自動車（EV）の普及に伴い、軽量化のニーズはますます高まると予想される。

今後は、以下の点に注力して開発を進めるべきである。

- **材料メーカーとの連携:** 低コストなCFRTP材料の開発
- **成形技術の確立:** ハイブリッド成形などの新しい成形技術の導入
- **シミュレーション技術の活用:** 設計・成形の最適化
