# 株式会社ファソテック・Visual Script Designer・AI設計代行に関する総合調査レポート

## エグゼクティブサマリー

機構設計分野におけるAI代替実現に向けて、日本の製造業DX支援企業である株式会社ファソテックとダッソー・システムズのVisual Script Designer (GGR)を中心とした調査を実施した。**ファソテックは40年以上の歴史を持つCAD/CAM/PLM分野のリーディング企業で、AI技術の積極導入により次世代モノづくりソリューションへの進化を図っている**。GGRは2023年リリースの革新的ノーコードツールとして、従来の視覚的プログラミングを超えたエンタープライズレベルの統合ソリューションを提供する。AI設計代行においては、ロジックツリー形式での設計手順構造化が非言語的ナレッジのAI理解可能形式への変換における重要な要素となる。

## 1. 株式会社ファソテック企業概要

### 企業基盤と事業展開

**株式会社ファソテック**は1983年設立の製造業支援企業で、千葉県幕張テクノガーデンに本社を構える。資本金260万米ドル、従業員194名の規模で、**日本で3次元形状をコンピュータシステムで実現したNC加工の草分け的存在**として業界をリードしてきた。同社の社名「FASOTEC」は「FAのSOftwareとTEChnologyに貢献する」という企業ミッションを体現している。

### 主力事業と技術的強み

**CAD/CAM/PLM統合ソリューション**を核として、ダッソー・システムズ製品（CATIA、3DEXPERIENCE）の販売・技術支援で市場をリードする。特筆すべきは**2020-2021年にMarkforged製品の日本市場販売実績No.1**を達成した3Dプリンティング事業で、カーボンファイバー・金属対応の高度な造形技術を提供している。

医療分野では独自開発の**外科手術シミュレーションシステム「FasoLab」**により、ダ・ヴィンチ等のロボット支援手術トレーニングを支援し、製造業で培った3Dデジタル技術の医療応用で新たな価値を創造している。

### AI技術への戦略的取り組み

ファソテックのAI導入は多層的なアプローチを採用している。**2020年12月に導入したKUZENとのAPI連携AIチャットボット**は、Q&Aデータと翻訳データベースを連携させた多言語自動応答システムを構築した。2025年の3DEXPERIENCE CONFERENCE JAPANではサファイヤスポンサーとして参加し、「AIが加速するバーチャルツインの可能性」をテーマとした技術革新をリードしている。

技術面では**Markforged BlacksmithのAI品質管理システム**を活用し、レーザーマイクロメーターによるリアルタイム計測とAI自動解析により造形品質の向上を実現している。これらの取り組みは、従来の製造業支援企業からAI・デジタル技術を活用した次世代ソリューションプロバイダーへの戦略的転換を示している。

## 2. Visual Script Designer (GGR) 技術詳細分析

### 革新的ノーコードプラットフォームの登場

**CATIA Visual Script Designer (GGR)**は、ダッソー・システムズが2022年11月の3DEXPERIENCE R2023xでリリースした画期的な視覚的プログラミングツールである。GGRは**ノーコードアプローチによるパラメトリックデザイン**を実現し、プログラミング知識不要でグラフノードベースインターフェースを通じて複雑な設計アルゴリズムを構築できる。

### R2024x技術革新の詳細

2024年の主要アップデートでは**AI支援オペレーター提案機能**が追加され、作業中のアルゴリズムを認識して適切なノードを自動提案する。Python統合により従来のノードに加えてPythonスクリプトの組み込みが可能となり、技術者の創造性と生産性を大幅に向上させている。

**Design Variants機能**では同一デザインの複数バリエーション管理が可能となり、**3D直接操作モディファイア**によりビューポート内での直接的パラメータ操作を実現している。これらの機能により、**7000m²超の内装ファサード自動生成**などの大規模プロジェクトでの実績を積み上げている。

### 競合ツールとの差別化戦略

GGRは既存の視覚的プログラミングツールに対して明確な差別化を図っている。**Grasshopper**が成熟したエコシステムと高速計算処理で優位性を持つのに対し、GGRは**PLM環境での完全統合とエンタープライズレベルのデータ管理**で対抗する。**Dynamo**の建築業務特化に対し、GGRは**産業用途での幅広い適用性とクロスプラットフォーム統合**を強みとする。

最も重要な差別化要因は**3DEXPERIENCE Platform**との完全統合であり、設計からシミュレーション、製造まで一貫したデータフローを実現し、大規模組織での協調設計を可能にしている点である。

## 3. AI設計代行時代における設計手順構造化戦略

### 現状のAI代替進展状況

機構設計分野でのAI導入は**段階的進展モデル**に従って発展している。現在の初期段階では反復的タスクの自動化（図面作成、制約設定、標準部品検索）が主流で、2025-2027年の中期段階では設計最適化、生成設計、予測メンテナンスへと発展する。2028年以降の長期展望では**包括的設計意思決定支援システム**の実現が期待される。

主要CADベンダーの戦略を見ると、**Autodesk**はFusion 360でのジェネレーティブデザインとAutoConstrainによるスケッチ制約自動化、**Dassault Systèmes**は3D UNIV+RSESとバーチャルツイン体験、**Siemens**はNX Design Copilotによる自然言語設計支援を展開している。

### 非言語的ナレッジ変換の技術的アプローチ

設計における暗黙知をAI理解可能形式へ変換する手法として、**4つの主要アプローチ**が確立されている。**論理的表現**では命題論理・述語論理を活用したIF-THENルールベースシステム、**セマンティックネットワーク**では部品間の階層関係をグラフ構造で表現、**フレームベース表現**では設計要素の属性集合と継承関係による知識階層化、**生産規則システム**では条件-行動ルールによる設計判断プロセスの手順化を実現する。

実装面では**マルチモーダル学習**による3D形状データと設計文書の同時学習、**知識グラフ構築**による設計パターンの関係データベース化、**オントロジー工学**による設計ドメインの概念体系化が重要となる。

### 3D CAD対コードベース駆動設計の戦略的比較

従来型3D CADは直感的な視覚的操作と創造性の直接反映で優位性を持つが、手作業による反復作業と設計変更の影響範囲把握困難という課題を抱える。**コードベース設計**は**OpenSCAD、FreeCAD Python Scripts、CadQuery**等のツールにより、パラメトリック設計の高度制御と設計ロジックの明示化を実現するが、プログラミング学習コストの高さが障壁となる。

**ハイブリッドアプローチ**として、**Zoo Design Studio**のテキストプロンプトからの3Dモデル生成、**DraftAid**の3Dモデルから2D図面への自動変換、**MecAgent**の文脈理解型部品生成など、視覚的設計とコードベース制御を融合した統合型プラットフォームが台頭している。

## 4. ロジックツリー形式設計手順構造化の実装戦略

### 構造化設計プロセスの戦略的利点

ロジックツリー形式での設計手順構造化は、**3つの核心的利点**を提供する。**構造化された意思決定プロセス**により設計選択肢の体系的整理とリスク評価の可視化を実現し、**トレーサビリティの確保**で設計変更の影響追跡と品質問題の根本原因分析を可能にする。**知識の体系化**では設計ノウハウの構造化と組織知識の継承を支援する。

### 階層型実装フレームワーク

実装手法として**3層構造の階層型設計決定システム**を提案する。レベル1では概念設計決定、レベル2では詳細設計パラメータ、レベル3では製造・組立考慮事項を扱う。具体的なフローとして「設計要求 → 材料選択 → 製造方法 → 形状最適化 → 評価」の各段階で「強度/重量」「加工性/コスト」「性能/制約」の分岐判断を構造化する。

AI統合では**機械学習による最適パス予測**、設計パターン認識、自動分岐条件生成を組み込み、**Python scikit-learn、R rpart、TensorFlow Decision Forests**等のフレームワークを活用する。

### 実用的応用事例とツール展開

最新のAI応用ツールとして、**Leo AI**は100万以上の技術資料に基づく設計支援で10,000以上のエンジニアが利用し、**MecAgent**は主要CADプラットフォーム統合で40%の作業時間削減を実現している。**Leaf Automation**は強化学習技術を活用した電気・機械システム設計自動化を提供する。

産業別では自動車産業でのジェネレーティブデザイン軽量化、航空宇宙での複雑形状最適化、再生可能エネルギーでの風力発電機翼設計最適化など、具体的な成功事例が蓄積されている。

## 5. パラメトリック・ジェネレーティブデザインの戦略的進化

### AI強化パラメトリック設計の技術革新

従来のパラメトリック設計から**AI強化パラメトリック設計**への進化が加速している。機械学習による最適パラメータ予測、多目的最適化の自動実行、設計性能の実時間評価により、設計プロセスの根本的変革を実現している。

ジェネレーティブデザインでは**トポロジー最適化**、進化的アルゴリズム、深層学習GANsを活用した設計候補生成が主流となり、**Autodesk Dreamcatcher、Siemens NX、PTC Creo、Parametric Hektar**等のプラットフォームが建築・都市計画から製品設計、構造工学、バイオミメティクスまで応用領域を拡大している。

### 2024年技術トレンドと将来展望

**マルチフィジックス統合**により熱・構造・流体解析の統合最適化とリアルタイム性能評価、持続可能性指標の組み込みが実現している。**AI駆動設計探索**では大規模パラメータ空間の効率的探索と設計制約の自動学習、人間の設計意図理解が進展し、**クラウドネイティブ化**によりスケーラブルな計算資源活用とリアルタイム設計検証が可能となっている。

## 戦略的実装ロードマップ

### 短期戦略：基盤構築期（1-2年）

**基盤技術の習得**としてAIツールの段階的導入、設計者のAIリテラシー向上、データ管理体制の整備を実施する。**パイロットプロジェクト**では限定的な設計タスクでのAI活用、ROI測定と効果検証、ベストプラクティス蓄積を推進する。

### 中期戦略：統合展開期（3-5年）

**統合プラットフォーム構築**でAI機能統合CAD環境、知識ベース管理システム、設計意思決定支援システムを構築する。**組織変革**では AI活用型設計プロセス標準化、人材育成プログラム拡充、部門間連携強化を実現する。

### 長期戦略：自律化実現期（5-10年）

**自律設計システム**でエンドツーエンド設計自動化、創造的AI設計パートナー、継続学習システムを構築する。**エコシステム構築**によりサプライチェーン統合、オープンイノベーション推進、業界標準化リーダーシップを確立する。

## 結論と戦略的提言

機構設計分野におけるAI代替は完全な代替ではなく**「人間+AI」の協働モデル**が現実解となる。株式会社ファソテックの40年の製造業支援経験とGGRの革新的ノーコード技術は、この協働モデル実現の重要な要素である。ロジックツリーによる設計手順構造化は、非言語的設計ナレッジをAI理解可能形式へ変換する核心技術として位置づけられる。

**成功の鍵は技術導入と人材育成の並行実施**、段階的なデジタル変革の推進にある。早期のAI導入と継続的な学習により、競争優位性の確保と持続可能な設計イノベーションの実現が可能となる。機構設計職能のAI代替実現に向けて、業務コンテキストのAI理解可能形式への変換とAI操作可能ツールへの代替を戦略的に推進することが重要である。
