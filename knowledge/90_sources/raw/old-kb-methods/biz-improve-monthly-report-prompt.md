# 月次業務レポート作成プロンプトテンプレート

このテンプレートは、メモ書きや日報から月次業務レポートを作成する際に活用できるプロンプトです。Rimoの「要約レポート作成の5STEP」に基づいて構成されています。

## 🎯 プロンプトの目的
- メモ書きから業務内容を体系的に抽出・整理する
- 読み手にとって分かりやすく、業務の流れや成果が伝わるレポートを作成する
- 所感や学び、今後の課題も含めて記録する

## 🧠 プロンプト構成（5STEP）

### 1. 全体像の把握
- メモの期間、担当者、業務カテゴリを抽出
- 業務の目的や背景を簡潔に記述

### 2. 業務内容の整理
- 各業務カテゴリごとに、実施内容・関係者・成果を記述
- 箇条書きで要約を追加

### 3. 他者の発言・引用の整理
- メモ内の発言者・引用内容を抽出
- 意見や指示の要点をまとめる

### 4. 所感・学び・失敗の記録
- 担当者の気づきや学び、改善点を記述
- 失敗事例とその対応策も記録

### 5. 構造化されたレポートの出力
- Markdown形式で出力
- 必要に応じてPDFやスライド形式に変換

## 🧾 使用例プロンプト

あなたは出版社で働く優秀な編集者です。
以下をもとに、月次業務レポートを作成してください：
- メモ期間：2025年7月3日〜7月18日
- 担当者：遠藤真輝
- 業務カテゴリ：試作発注、設計図面、解析評価、ナレッジ管理、会議設定、その他
- 各業務の背景、目的、成果を明記
- 他者の発言を整理
- パスワードや専門用語についても整理
- 所感・学び・失敗を記述
出力形式はMarkdownとし、構造化されたレポートとしてまとめてください。
