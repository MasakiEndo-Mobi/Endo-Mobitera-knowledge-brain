# 【遠藤・水野】業務確認 議事録

**日時:** 2025年8月20日 | **時間:** 開始時刻不明 - 終了時刻不明 | **所要時間:** 59分

## 基本情報

**参加者:**

- 遠藤 真輝 (ENDO MASAKI) - 担当者
- 水野 涼太 (MIZUNO RYOTA) - 上級担当者

**会議の目的:** DR0資料作成進捗確認と業務調整

## 議事内容

### 1. DR0資料作成と進捗管理 [00:02:21]

**議論内容:**

- DR0資料の段階的作成手順の確認
- 3番まで完成時点での確認フロー設定
- 修正効率化のための一気通貫作成方針

**決定事項:**

- **作成手順**: 3番まで完成後に水野さん確認、その後桐生さん・佐藤さんレビュー
- **納期管理**: 9月25日最終期限、9月10日仮提出期限
- **優先順位**: DRBが最も時間を要するため最優先で進行

**必要資料一覧:**

- DRB（デザインレビューブック）
- 信頼性試験計画書
- 設計基準チェックリスト
- カコトラチェックリスト
- 品質情報
- 安全性レポート
- FMA（故障モード解析）

### 2. 信頼性試験計画と評価項目整理 [00:22:03]

**議論内容:**

- 過去機種（695、155等）の事例参照
- 気候・電気部門の役割分担明確化
- 社内外合意形成プロセスの確認

**決定事項:**

- **ベース**: 1S実施内容に熱衝撃等を追加
- **気候評価**: 振動性能、振動耐久、衝撃強度等
- **電気評価**: 電気部門が必要に応じて実施
- **合意プロセス**: 社内信頼性チーム合意後、顧客説明・合意

**評価項目詳細:**

- 対象内：振動性能、振動耐久、衝撃強度、ナトルカ単振動
- 対象外：耐水性、耐油耐薬、操作耐久、特殊操作（構造踏襲・実績考慮）

### 3. 設計基準・カコトラチェックリスト運用 [00:08:26]

**議論内容:**

- 最新フォーマット（2023年版）の確認
- 他機種事例参照方法
- 対象外項目の判断基準設定

**決定事項:**

- **設計基準**: 2023年から変更なし、M・S項目中心に確認
- **カコトラ**: 5月最新化版使用、該当・非該当理由明記
- **判断基準**: 過去実績・構造踏襲の有無で対象外項目決定

### 4. 両面テープ関連工場確認 [00:53:30]

**議論内容:**

- ディスカッション穴あけ作業の可否確認
- セパレーター位置決め方法の検討
- 大連工場での作業可能性調査

**決定事項:**

- 工場・大連・松本石器等関係先への確認実施
- 情報を伏せた状態での参考確認アプローチ
- 新形状の早期発注とアーク納期調整

### 5. 二年目研修振り返りと業務サポート [00:46:05]

**議論内容:**

- 自己課題の言語化（自走力不足、失敗経験少）
- フレームワーク活用となぜなぜ分析の実践
- 継続的成長に向けたサポート体制構築

**決定事項:**

- **定例**: 週1回水曜日12時の業務整理ミーティング
- **柔軟対応**: 必要時30分追加枠設定
- **目標**: 3-6ヶ月後の自走可能状態達成

## アクションアイテム

- [ ] カコトラ・設計基準チェックリスト作成 - 遠藤 - 今週中（桐生さん・佐藤さん確認まで）
- [ ] DRB資料作成（3番まで完成後レビュー依頼） - 遠藤 - 9月10日まで
- [ ] 信頼性試験計画表作成・電気部門すり合わせ - 遠藤 - 9月25日まで
- [ ] 両面テープ工場・大連確認（タブ作業可否） - 遠藤 - 早急
- [ ] 両面テープ発注・納期調整（アーク連携） - 遠藤 - 早急
- [ ] 板金図面進捗確認 - 遠藤 - 8月25日まで
- [ ] 川崎さんメッセージ追加依頼 - 遠藤 - 即日
- [ ] 野崎さんファン禁止エリア情報共有 - 水野 - 即日
- [ ] 週1業務整理ミーティング設定 - 水野・遠藤 - 継続実施

## 未解決事項

### 両面テープタブ位置決め確認

**課題:** ディスカッション部穴あけ作業の可否と必要穴径、大連工場での作業可能性 **対応方針:** 開発情報を伏せた参考確認として関係先に問い合わせ

### 信頼性試験計画の電気部門調整

**課題:** 気候・電気部門の試験項目分担と実施体制の明確化 **対応方針:** 早期に電気部門とミーティング設定し項目すり合わせ実施

## 決定事項まとめ

1. **DR0資料は段階的作成で効率化、9月10日仮提出・25日最終完成**
2. **DRBを最優先で進行、他資料は並行作業**
3. **信頼性試験は過去実績ベースで計画、電気部門と早期調整**
4. **チェックリスト類は最新フォーマット使用、今週中に上位者確認**
5. **両面テープは工場確認後に発注、納期調整も並行実施**
6. **週1回の業務サポート体制で遠藤さんの自走力向上支援**
