# 24CY 744D 機構DP 2回目 要約

## 文書情報
- **文書タイトル**: 24CY 744D 機構DP 2回目
- **発行元**: Panasonic AUTOMOTIVE
- **機密区分**: Confidential
- **ページ数**: 8ページ
- **文書種類**: 技術設計レビュー資料（デザインポイントレビュー）

## エグゼクティブサマリー

本資料は、24CY 744D機構設計の2回目のデザインポイント（DP）レビュー資料である。前回DP（2024年6月6日）以降に発生した顧客仕様変更3点に対する設計対応と技術的課題の検討内容を詳細に説明している。主な変更は、温調照明のブラックアウト化、ボタン意匠のブラックアウト化、およびノブ意匠の24CY400Dとの共通化である。

## 主要な仕様変更内容

### 1. 温調照明追加（ブラックアウト仕様）
- 温度コントロールシンボルに照明を追加し、ブラックアウト仕様化
- ロゴマーク（JBL、PWR、VOL、nanoeX）は照明なし、非ブラックアウト

### 2. ボタン意匠ブラックアウト仕様化
- 表面からの見栄えをピアノブラック相当に変更
- ボタンのシンボルとインジケータをブラックアウト化（788Dと同仕様）

### 3. ノブ意匠24CY400Dと共通化
- フィーリングはレクサス仕様を維持
- 外観意匠を400Dと統一

## 各変更点の詳細設計対応

### 1. 温調照明追加の技術的対応

#### デザインポリシー
- レジスターパネルを**2色成形化**でブラックアウト実現
  - 1次成形：黒色材料
  - 2次成形：クリア材料
- 表面にクリア塗装とロゴマーク印刷を実施

#### 技術的課題と対策

| 課題 | 対策 |
|------|------|
| 2色成形の肉厚確保 | 板厚2.4mm設定で対応可能（実績：21CY960Bで3mm） |
| 照明のけられ防止 | LEDをシンボル真裏配置、1次形状にC面取り |
| 材料収縮率差による剥離 | PC/ABS組合せ実績あり（線膨張係数を考慮） |
| 隣接ボタンとの見栄え統一 | 同一材料採用、天肉1.5mmで統一 |

### 2. ボタン意匠ブラックアウト化の技術的対応

#### デザインポリシー
- 788Dの釦構成をキャリーオーバー
  - キャップ：スモークPC
  - 裏面から塗装
  - シンボルをレーザー抜き
  - 照明ムラ防止用ライトガイド
- 釦間の桟：ESC形状作成＋金型磨きでピアノブラック調

#### 技術的課題と対策
- **桟の見栄え**: 幅0.83mmで目立ちにくい設計
- **塗装指示リスク**: 現時点で顧客からの明確な塗装指示なし

### 3. ノブ意匠共通化の技術的対応

#### デザインポリシー
- **部品流用戦略**
  - ライトガイドアウターアッシー：310D流用
  - 両面テープ：310D流用
  - ノブトップ：310D流用
  - ノブベース：新設（レクサスエンコーダー対応）

#### 技術的課題と対策

| 課題 | 対策 |
|------|------|
| エンコーダー嵌合隙間 | 一部減肉で円周方向0.75mm、押込方向1mm確保 |
| 嵌合力確保 | 試作確認で抜去力14.7N以上を達成 |
| 薄肉部ショートショット | サプライヤーとゲート位置調整 |
| 照明への影響 | T系エンコーダーでシャフト外形有利 |

## その他の検討事項

### トグル部の設計検討
- **基本方針**: 867Dの構成をキャリー（レバー比0.43）
- **課題**: レクサスフィーリングの準拠
- **対応状況**:
  - トヨタ車体へ試作サンプル確認依頼中
  - 744D車両担当者への確認連絡済み
  - 領域拡大の可否について確認中

## 今後の対応事項

1. **材料メーカーへのヒアリング**
   - 大物部品での2色成形可否確認
   - PC/ABS組合せの信頼性確認

2. **顧客確認事項**
   - トグル部フィーリングのサンプル確認
   - 桟部分の仕上げ仕様の最終確認

3. **サプライヤー調整**
   - ゲート位置の最適化
   - 金型仕上げ仕様の確定

## まとめ

本DPレビューでは、顧客要求の3つの仕様変更に対して、既存部品の流用と新規設計を組み合わせた合理的な対応策を提示した。特に、ブラックアウト化については2色成形技術の活用、ノブ共通化については310D部品の最大活用により、品質とコストの両立を図っている。今後は、材料メーカーとの詳細確認と顧客承認を経て、量産設計へ移行する予定である。
